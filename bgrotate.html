jQuery Background Rotator Example

This page demonstrates my very simple jQuery plug-in that rotates a sequence of background images, using a delay, on a DOM element.
Because it's ancient it works with really old browsers and jQuery versions - but also works with newer ones, too :)

This page has 3 examples. Two DIV elements and the BODY element. The code to call them is shown below 

$(document).ready(function()
{
	$("body").bgrotate({ delay: 4000, imagedir: "/images/backgrounds/", images: ["249.jpg", "335.jpg", "419.gif", "87.jpg", "96830.png"] });
	$("#div1").bgrotate({ delay: 800, imagedir: "/images/backgrounds/", images: ["pattern8-pattern-58a.png", "419.gif"] });
	$("#div2").bgrotate({ delay: 1200, imagedir: "/images/backgrounds/", images: ["Rust_Peel.jpg", "WetPaint2.jpg", "Old_Dollhouse.jpg"] });
});
